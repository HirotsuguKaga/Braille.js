tactileGraph.js

tactileGraph.js is a canvas like tactile graphic drawing library.

Download this project as a .zip file Download this project as a tar.gz file

概要

tactileGraph.jsはブラウザ上で点図を生成し、印刷用のPNG画像やエーデルファイルを出力するJavaScriptライブラリです。
