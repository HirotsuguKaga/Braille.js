About tactileGraph.js

tactileGraph.js is a canvas like tactile graphic drawing library.

Download this project as a .zip file Download this project as a tar.gz file

詳しい使い方

  1. 以下のリンクからZIPファイルをダウンロードして解凍します。
    　tactileGraph.js.zip

  2. 解凍したフォルダから"tactileGraph.js"か"tactileGraph.min.js"のどちらかのjsファイルを選択してhtmlファイルと同じフォルダに置きます。
    （二つのファイルの中身は基本的に同じものですが、後者は可読性を無視して改行やコメントを取り除いてあり、その分だけファイルサイズが小さく、動作が若干早くなっています）

  3. htmlファイルの他のJavaScriptコードよりも上部に選択したjsファイルへのリンクを書き込みます。リンク文は、
    <script src="tactileGraph.js"></script>もしくは
    <script src="tactileGraph.min.js"></script>となります。

  4. 以降はJavaScriptのコードの説明になります。
    最初に任意の変数に"tactileGraph"を代入し、点図オブジェクトを作成します。
    　ex.　var tg = tactileGraph; //「tg」の部分は任意
    プログラム内には、作成した点図オブジェクトの数だけ点図データが保持されます。

  5. 作成したオブジェクトの名前を「苗字」として各メソッドを実行します。
    以下は、点図の画面上のプレビューに用いるhtmlのCanvas要素のIDを設定するメソッドの例です。（プレビューを必要としないプログラムであればcanvas要素は不要です）
    　ex.　tg.setCanvas("cvs");

  6. 座標を指定して描画メソッドを実行することによって点図を作成していきます。
    各メソッドの詳しい説明はREADME_ja.mdファイルにあります。
    　ex.　tg.drawBraille("ABCDEF", 0, 10);
    　　　tg.drawLine(0,20,50,20);

  7. 作成した点図データを出力するには、点図データを返り値としたメソッドをダウンロードリンクなどに代入します。
    　ex.　download.href = tg.map2esa();

エーデルについて

エーデルとは、当時、盲学校教師だった藤野稔寛氏が開発した図形点訳ソフトです。
最新版のエーデルは以下のサイトから無料でダウンロードできます。
　EDEL-plus のホームページ
エーデルで描いたものは点図プリンタで打ち出したり、あるいは墨字印刷をして立体コピーに利用されています。
エーデルで作成したファイルは「 .edl」という拡張子で保存されます。 エーデルには対応する点字プリンタに応じていくつかのバリエーションが存在します。

用紙サイズについて

現在、本ライブラリで作成できるのは、エーデル8が対応する10×11インチ用紙（通称、A4用紙）と8×10インチ用紙（通称、B5用紙）の二種類です。
